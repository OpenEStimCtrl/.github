# OpenEStimCtrl - 开放的电刺激控制协议库

## 支持的设备
- [x] DG-Lab ESTIM01 地牢实验室 郊狼1代
- [x] YokoNex ES01 役次元1代

## 提供的功能
- [x] 构建 BLE 发送数据包
- [x] 解析设备的 BLE 回应并通过事件函数返回
- [x] 自动端序转换
- [x] 波形生成计算
